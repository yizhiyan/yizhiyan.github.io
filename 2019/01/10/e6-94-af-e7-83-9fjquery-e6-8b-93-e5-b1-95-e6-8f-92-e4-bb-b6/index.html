支烟jquery拓展插件

这是一个未完成的插件，可能以后来兴趣了可以继续更新吧，现在就是发出来吧

可以做什么？

zy={
image:{
rollingSeamless:左右无缝滚动
rollingFade:淡出淡入方式进行图片切换
},
attr:{
getMargin(id):获取margin
getPadding(id):获取padding
toTop(time,p):回到顶部,time是时间,p偏移距离顶部
},
equipment:{
detectOS
isIPhone
isAndroid
isMobilePhone
isWindow
isLinux
},
scroll:{
getScrollTop:取滚动条距离顶部位置
getClientHeight:取当前可视范围高度
getScrollHeight:获取文档完整的高度
prohibit(isStart):是否禁止显示滚动条 isStart=true禁止else true
reach(id, small, large):滚动条移动到当前dom位置时触发 small小于当前位置时触发 large大于或等于当前位置时触发 只触发一次
},
event:{
//滚动时触发
scroll
scrollUp
scrollLeft
scrollRight
scrollDown
scrollX
scrollY
scrollBottom:滚动到底部时触发
mouseClickMove:按下滑动触发
},
mobile:{
//rem操作,docWidth:设计稿宽度,bNumber:倍数越高则偏差越少
//minWidth最小宽度,maxWidth最大宽度
//默认：docWidth=750 bNumber=100 minWidth=320 maxWidth=默认：docWidth
rem(docWidth, bNumber, minWidth, maxWidth)
},
regular:{
z(z,s):正则匹配，z正则表达式 s匹配字符串
isNuber
isPhone
isQQNumber
isWeiXin
isEmail
isUrl
isMoney
isTime
isUnsignedNumeric
},
other:{
followScroll:卡片跟随
getRealSize:获取实际大小
}
}
